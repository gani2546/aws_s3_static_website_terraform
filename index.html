About the Project

This project showcases a scalable web application designed using AWS 3-tier architecture. It includes an application layer, a database layer, and a storage layer.

Features

  • High availability using Auto Scaling Groups and Load Balancers
  • Scalability with AWS EC2 and RDS
  • Secure and efficient storage with AWS S3

Technologies Used

  • AWS EC2
  • AWS S3
  • AWS RDS
  • Terraform
